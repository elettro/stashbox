Stashbox Radio React Preview

Stashbox Radio React Preview is a test version of the browser-based Stashbox streaming player at /radio/react/. It uses the same published Google Sheet/App Script song feed fields as the current Stashbox Radio page, including song title, album or collection, artist, genre, release date, audio URL, artwork URL, video link, notes, plays, and share counts.

The preview is designed for rock, reggae, blues, funk, soul, calypso, electronic, Spanish, pop, original songs, tribute tracks, videos, and song-related merch recommendations. Visitors can search songs, filter by genre, select a track, listen with browser audio controls, and open an available video without letting the audio player play over it.

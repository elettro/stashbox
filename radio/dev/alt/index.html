Stashbox Radio RDS Dev Alt

Stashbox Radio RDS Dev Alt is a safe alternate test version of the browser-based Stashbox streaming player at /radio/dev/alt/. It uses the AWS RDS API as the development database source while the production /radio/ page remains unchanged. The active song feed reads from the RDS songs endpoint and uses audio URLs, video links, resolved artwork, public notes, video notes, and song-specific product URLs.

The preview is designed for rock, reggae, blues, funk, soul, calypso, electronic, Spanish, pop, original songs, tribute tracks, videos, and song-related merch recommendations. Visitors can search songs, filter by genre, select a track, listen with browser audio controls, and open an available video without letting the audio player play over it.
